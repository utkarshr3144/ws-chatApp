# WebSocket Room Chat

A basic real-time chat application built using React, TypeScript, Node.js, and the native `ws` WebSocket library.

## Features

- Room-based chat
- Real-time messaging
- React frontend
- Node.js WebSocket backend
- TypeScript on frontend and backend

## Tech Stack

- React
- TypeScript
- Node.js
- WebSockets (`ws`)
- Vite
- Tailwind CSS

## Future Improvements

- Authentication
- Persistent chat history
- PostgreSQL + Prisma
- Better UI
- Redis Pub/Sub for scaling
